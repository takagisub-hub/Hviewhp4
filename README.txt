Hview 横スクロールHPラフ

■ 開き方
index.html をダブルクリックしてください。

■ FOOD画像
assets/food.jpg に、いただいた写真を設定済みです。

■ 写真の入れ替え方
1. assets フォルダへ新しい写真を入れる
2. index.html 下部の「const SLIDES」にある image を変更する

例：
image: "assets/fitness.jpg"

■ 写真の表示位置
position: "center 50%" を変更できます。

例：
position: "center"
position: "50% 35%"
position: "left center"

■ 構成
・外部ライブラリなし
・横ドラッグ／スワイプ対応
・左右ボタン対応
・下部カテゴリー選択対応
・スマホ表示対応
